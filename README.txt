- Install:
	Installeer Eclipse
	Installeer Android SDK (Beste is om alle APIs van Android 2.x en 4.x te installeren + alle tools)
	Installeer Android ADT Plugin
- Import/Setup:
	In Eclipse: Import->Android->Existing Android Code, importeer de project folder
	Facebook en tapjoy zouden automatisch mee moeten komen in Eclipse (zo niet voeg ze toe)
	Los eventuele errors op, veel kan worden verholpen door Project->Clean uit te voeren
- Export:
	Export->Android->Export Android Application
	Kies het untitled project (of hoe je hem ook hebt genoemd)
	Maak keys aan (simpel PW)
	Finish
- Running:
	Verplaats .APK naar je telefoon (bijv. dropbox of kabel)
	Installeer .APK
	Done

Aanpassingen kan je doen in de assets folder (daar zitten alle Lua/Moai related dingen)
Daarna Export en Running stappen opnieuw uitvoeren
